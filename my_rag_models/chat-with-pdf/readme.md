Chat with PDF using LangChain ang OpenAI

This project allows you to interactively chat with the contents of a PDF document using advanced language models. By leveraging [LangChain](https://langchain.com/) and [OpenAI](https://openai.com/), you can ask questions about your PDF and receive insightful answers directly based on its content.

**Features:**
- Upload any PDF and converse with it as if it were an expert on its own content.
- Uses LangChain for effective document chunking and retrieval.
- Employs OpenAI's models to generate intelligent, context-aware responses.
- Enables research, study, or information extraction from PDFs in a conversational way.

**Getting Started:**
1. Clone the repository.
2. Install the required dependencies listed in `requirements.txt`.
3. Run the application, upload your PDF, and start chatting!

This tool is perfect for students, researchers, and anyone who needs to quickly extract information from long PDF documents.
